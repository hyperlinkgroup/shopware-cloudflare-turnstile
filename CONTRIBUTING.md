# Mitwirken an SsqCloudflareTurnstile

Vielen Dank für dein Interesse an diesem Projekt! Wir freuen uns über Beiträge jeder Art – ob Bugfixes, neue Features, Dokumentation oder Tests.

---

## Ablauf

1. **Fork** des Repositories erstellen
2. **Feature-Branch** anlegen: `git checkout -b feature/mein-feature`
3. Änderungen vornehmen und testen
4. **Committen**: `git commit -m 'Feature: Kurze Beschreibung'`
5. **Pushen**: `git push origin feature/mein-feature`
6. **Pull Request** öffnen mit einer klaren Beschreibung der Änderungen

---

## Code-Qualität

- Halte dich an den bestehenden Code-Stil (PSR-12 für PHP)
- Stelle sicher, dass das Plugin nach deinen Änderungen weiterhin korrekt installiert, aktiviert und deinstalliert werden kann
- Teste deine Änderungen in einer lokalen Shopware-6.5-Instanz
- Achte darauf, dass sowohl die deutschen als auch die englischen Übersetzungen vollständig sind

---

## Einsatz von KI-Tools

Der Einsatz von KI-gestützten Tools (z. B. GitHub Copilot, ChatGPT, Claude etc.) zur Code-Erstellung ist **erlaubt**.

**Voraussetzung:** Jeder Beitrag – unabhängig davon, ob er mit oder ohne KI erstellt wurde – **muss vollständig von einem Menschen reviewed und verstanden worden sein**, bevor ein Pull Request eröffnet wird.

Das bedeutet konkret:

- Du verstehst jede Zeile des eingereichten Codes
- Du hast den Code in einer lokalen Umgebung getestet
- Du übernimmst die volle Verantwortung für den Beitrag

Pull Requests mit offensichtlich unreviewtem, generiertem Code werden abgelehnt.

---

## Bugs melden

- Nutze die [GitHub Issues](../../issues), um Bugs zu melden
- Beschreibe das Problem möglichst genau: Shopware-Version, PHP-Version, Schritte zur Reproduktion
- Füge relevante Log-Einträge bei (z. B. aus `var/log/`)

---

## Feature-Vorschläge

- Öffne ein Issue mit dem Label `enhancement`
- Beschreibe den gewünschten Mehrwert und einen möglichen Lösungsansatz

---

## Lizenz

Mit dem Einreichen eines Pull Requests stimmst du zu, dass dein Beitrag unter der [MIT-Lizenz](LICENSE) dieses Projekts veröffentlicht wird.
